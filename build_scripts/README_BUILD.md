# Скрипты сборки UTILHELP

Эта папка содержит скрипты для создания исполняемого файла UTILHELP и установщика.

## Требования

- **Python 3.8+** с PyQt6
- **PyInstaller** (`pip install pyinstaller`)
- **Inno Setup 6** для создания установщика ([Скачать](https://jrsoftware.org/isdl.php))

## Использование

### 1. Сборка программы
```bash
1_build.bat
```
Создает `dist/UTILHELP/UTILHELP.exe` со всеми зависимостями.

### 2. Создание установщика
```bash
2_installer.bat
```
Создает установщик `installer_output/UTILHELP_Setup_v1.0.exe`.

## Описание файлов

- **1_build.bat** - Основной скрипт сборки
- **2_installer.bat** - Скрипт создания установщика
- **build_clean.bat** - Процесс чистой сборки (вызывается из 1_build.bat)
- **build_installer.bat** - Сборщик установщика (вызывается из 2_installer.bat)
- **utilhelp_structured.spec** - Конфигурация PyInstaller
- **utilhelp_installer.iss** - Конфигурация Inno Setup
- **reorganize_build.py** - Реорганизация структуры после сборки

## Процесс сборки

1. **Очистка**: Удаляет старые файлы сборки и кэш Python
2. **Компиляция**: Использует PyInstaller для создания исполняемого файла
3. **Реорганизация**: Перемещает ресурсы в правильные места
4. **Упаковка**: Создает установщик с помощью Inno Setup

## Результат

- `dist/UTILHELP/` - Портативная папка программы
- `installer_output/UTILHELP_Setup_v1.0.exe` - Установщик Windows

## Примечания

- Скрипты сборки работают из корневой папки проекта
- Все пути относительны к основной папке проекта
- Установщик включает автоматическое определение обновлений
- Программа может работать без установки (портативный режим)
